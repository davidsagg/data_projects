## English

This is a **monorepo** gathering my *Build To Learn* projects — each one built end to end to deepen a specific engineering skill (data engineering, ML, LLMs, full-stack, automation). Every project lives in its own folder with its own `README`, docs and `CLAUDE.md` context file.

Each project's story is also written up as a case study on my Substack, **[David Saggioro — Case Studies](https://davidsaggioro.substack.com/)**, in both English and Portuguese.

### Projects

| Project | What it is | Stack highlights |
|---|---|---|
| [`bandkit`](bandkit/) | Show & setlist manager for gigging bands (PWA, live chord transposition) | FastAPI · React · SQLite · pdfplumber · custom ChordPro engine |
| [`crypto_advisor`](crypto_advisor/) | Weekly crypto swing-trade advisor with IR tax optimization | Python · LLM agent |
| [`ficadica`](ficadica/) | Authenticated scraper + personalized study plan for a guitar course platform | Playwright · pandas · local dashboard |
| [`job_scout`](job_scout/) | Freelance-job monitor with Claude scoring + Telegram alerts | Python · RSS · Anthropic API · APScheduler |
| [`musicdna-ai`](musicdna-ai/) | Audio analysis with embeddings, licensing match & jam simulation (7 TDD phases) | Python · vector DB · Streamlit |
| [`Preditiva`](Preditiva/) | ML/stats case studies (Diabetes, HR turnover, Value at Risk) | pandas · scikit-learn · notebooks |
| [`saggirag`](saggirag/) | RAG chat over finance content with Mistral 7B | RAG · Mistral · FastAPI |
| [`special_gear`](special_gear/) | Monitor of Brazilian federal auctions for music/audio gear | Python · scraping · Telegram |
| [`trend-radar`](trend-radar/) | Brazilian music-market trend intelligence platform | Airflow · dbt/DuckDB · ML · local LLM |
| [`velodna`](velodna/) | Cycling performance platform | data pipeline · analytics |
| [`work_in_progress`](work_in_progress/) | Incubator for scripts/models under development | mixed |

> Repository conventions and per-project status live in [`CLAUDE.md`](CLAUDE.md).
